# Space Invaders

A collection of Space Invaders game implementations created with different AI assistants and tools.

## Projects

### spaceinvader-ClaudeCode
A Space Invaders implementation built with Claude Code. Features include:
- Interactive gameplay with keyboard controls
- Enemy waves and scoring system
- Collision detection
- Web-based game using HTML5 Canvas

**Files:**
- `index.html` - Main game page
- `tests.html` - Test suite
- `src/game.js` - Core game logic
- `src/player.js` - Player mechanics
- `src/enemies.js` - Enemy management
- `src/utils.js` - Utility functions

## Getting Started

Each project is self-contained. To play a game:

1. Open the `index.html` file in a web browser from the respective project folder
2. Use arrow keys or WASD to move your ship
3. Use spacebar to shoot
4. Destroy all enemies to advance to the next wave

## Development

Each project folder contains:
- Documentation in `docs/` folder
- Source code in `src/` folder
- Game assets and configuration files

For detailed gameplay information, see `GAMEPLAY.md` in each project's docs folder.
